# Emoji-Builder
Convert sentence to emoji-prefix-attached one.

## Example

### Input

|  Prefix  | Sentence |
|----------|----------|
| hym_big_ | Accrroad |

### Output

```text
:hym_big_a:​:hym_big_c:​:hym_big_c:​:hym_big_r:​:hym_big_r:​:hym_big_o:​:hym_big_a:​:hym_big_d:
```
